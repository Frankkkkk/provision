drpcli tasks show
=================

Show a single task by id

Synopsis
--------

This will show a tasks.

You may specify the id in the request by the using normal key or by
index.

Functional Indexs:

-  Name = string

When using the index name, use the following form:

-  Index:Value

Example:

-  e.g: Name:fred

::

    drpcli tasks show [id] [flags]
